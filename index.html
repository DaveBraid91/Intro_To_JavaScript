Variables

Una variable en programación es un espacio de memoria que almacena un valor que puede cambiar durante la ejecución de un programa. Las variables tienen un nombre que las identifica y un tipo que determina el rango de valores que pueden almacenar y las operaciones que se pueden realizar con ellas. Las variables se pueden declarar explícitamente o inferirse implícitamente según el lenguaje de programación que se utilice.

Palabras clave para declarar una variable en JavaScript

  • var:
    • Es la forma más antigua de declarar variables en JavaScript. Tiene un alcance global o de función, lo que significa que la variable es visible en todo el código o dentro de la función donde se declara. Además, permite redeclarar y reasignar el valor de la variable sin problemas. Sin embargo, esto puede causar problemas de sobrescritura o colisión de nombres, especialmente en código grande o complejo. Por eso, se recomienda evitar el uso de var y preferir let o const.
  • let:
    • Es una forma más moderna y segura de declarar variables en JavaScript. Tiene un alcance de bloque, lo que significa que la variable solo es visible dentro del bloque donde se declara (por ejemplo, un bucle, una condición o una función). Además, no permite redeclarar la variable con el mismo nombre en el mismo bloque, pero sí permite reasignar su valor. Esto evita los problemas de var y ofrece más control sobre el uso de las variables.
  • const:
    • Es otra forma más moderna y segura de declarar variables en JavaScript. Al igual que let, tiene un alcance de bloque y no permite redeclarar la variable con el mismo nombre en el mismo bloque. Sin embargo, a diferencia de let, no permite reasignar el valor de la variable una vez que se ha inicializado. Esto significa que la variable debe tener un valor asignado al momento de su declaración y ese valor no puede cambiar. Esto es útil para declarar constantes o valores que no deben modificarse.

Tipos de variables primitivas en JavaScript

  • Números
    • Números enteros
    • Números con decimales
    • NaN (Not a number): Números que no existen (Ej: 0/0)
  • Booleanos
    • Pueden tomar los valores "true" o "false".
  • Cadenas de caracteres (string)
    • Un String es un tipo de dato que representa una secuencia de caracteres. En JavaScript, los Strings se pueden crear usando comillas simples (') o dobles ("), o usando el acento grave (`).
  • Null
    • Null es un valor especial que representa la ausencia o la falta INTENCIONAL de un valor. Es uno de los valores primitivos de JavaScript. Null se puede asignar a una variable o a una propiedad de un objeto para indicar que no tiene ningún valor.
  • Undefined
    • Undefined es un valor especial que indica que una variable no tiene ningún valor asignado. También se usa como el valor predeterminado de las propiedades de los objetos que no existen, de los parámetros de las funciones que no se pasan, y del resultado de las expresiones que no devuelven ningún valor.

Operadores en JavaScript

  • Suma (+)
    • Da como resultado la suma de los elementos a su izquierda y a su derecha.
    • Se puede usar para concatenar cadenas de caracteres
    • Ejemplo: 3 + 2 da como resultado 5
    • Ejemplo: "Hola, " + "¿Qué tal?" da como resultado "Hola, ¿Qué tal?"
  • Resta (-)
    • Da como resultado la resta del número a su izquierda menos el de su derecha.
    • Ejemplo: 3 - 2 da como resultado 1
  • Multiplicación (*)
    • Da como resultado la multiplicación de los elementos a su izquierda y a su derecha.
    • Ejemplo: 3 * 2 da como resultado 6
  • División (/)
    • Da como resultado la división del número a su izquierda entre el de su derecha.
    • Ejemplo: 3 % 2 da como resultado 1.5
      (Recuerda que en Js el signo decimal es un punto y NO una coma).
  • Módulo (%)
    • Da como resultado el resto de la división del número a su izquierda entre el de su derecha.
    • Ejemplo: 3 % 2 da como resultado 1
    • Recuerda que el resto de una división sólo puede dar resultados entre 0 y el divisor - 1 y que, si el dividendo es múltiplo del divisor, el resultado será 0.
    • Ejemplo: n/60 puede dar resultados entre 0 y 59.
  • Exponente (**)
    • Da como resultado número a su izquierda elevado al de su derecha.
    • Ejemplo: 3 ** 2 da como resultado 9

Operadores sobre una variable en JavaScript

  • Sumar/Restar 1 post-valor (i++/i--)
    • Da como resultado el valor actual de la variable y después le suma/resta uno.
    • Ejemplo: variable = 3, variable++ da como resultado 3, pero ahora en variable hay guardado un 4.
  • Sumar/Restar 1 pre-valor (++i/--i)
    • Suma/resta uno al valor de la variable y devuelve el resultado.
    • Ejemplo: variable = 3, variable++ da como resultado 4.
  • Suma (+=)
    • Suma a la variable el resultado de la operación a la derecha del "=".
    • Se puede usar para concatenar cadenas de caracteres
    • Ejemplo: Si n = 3, tras n += 2 hay un 5 en n.
    • Ejemplo: Si n = "Hola, ", tras n += "¿Qué tal?" hay un "Hola, ¿Qué tal?" en n.
  • Resta (-=)
    • Resta a la variable el resultado de la operación a la derecha del "=".
    • Ejemplo: Si n = 3, tras n -= 2 hay un 1 en n.
  • Multiplicación (*=)
    • Multiplica la variable por el resultado de la operación a la derecha del "=".
    • Ejemplo: Si n = 3, tras n *= 2 hay un 6 en n.
  • División (/=)
    • Divide la variable entre el resultado de la operación a la derecha del "=".
    • Ejemplo: Si n = 3, tras n /= 2 hay un 1.5 en n.
  • Módulo (%=)
    • Divide la variable entre el resultado de la operación a la derecha del "=" y guarda el resto de dicha división en la variable.
    • Ejemplo: Si n = 3, tras n %= 2 hay un 1 en n.

Strings (Cadenas de caracteres)

  • Formas de crear Strings:
    • Comillas simples (')
    • Ejemplo: let myString = 'Dijo: "Hola, ¿Qué tal?"'
    • Comillas dobles (")
    • Ejemplo: myString = "Dijo: 'Hola, ¿Qué tal?'"
    • Acento grave (`)
    • Ejemplo: let myString = `Dijo: "What's up, hommie?"`
  • Índice de los caracteres en los Strings
    • Cada caracter de una cadena de caracteres tiene asociado un índice.
    • El primero tiene el íncide 0 y el último tiene índice longitud - 1.
    • Ejemplo: En myString el caracter "D" tiene el índice 0 y el caracter "'" tiene el índice 22 (23 - 1).
    • let myChar = myString[7] guarda "H" en myChar
  • Plantillas literales
    • Las plantillas literales son cadenas literales que permiten el uso de expresiones incrustadas
    • Sólo se pueden usar entre acentos graves
    • Ejemplo: myString = `He contado ${3 + 4} ovejas.`
      guarda en myString "He contado 7 ovejas."
  • Métodos de String
    • Cambio de mayúsculas y minúsculas:
      • toLowerCase() devuelve el string con todas las letras minúsculas.
      • Ejemplo: myString.toLowerCase() devuelve "dijo: 'hola, ¿qué tal?'"
      • toUpperCase() devuelve el string con todas las letras mayúsculas.
      • Ejemplo: myString.toUpperCase() devuelve "DIJO: 'HOLA, ¿QUÉ TAL?'"
    • Buscar por índice (indexOf(arg)):
      • myString.indexOf(arg) devuelve el índice del primer elemento del string guardado en arg, si se encuentra en myString
      • Ejemplo: myString.indexOf('hola') devuelve un 7
      • Si no encuentra el string guardado en el argumento, devuelve un -1
    • Trocear Strings (slice(arg) y slice(arg1, arg2))
      • myString.slice(arg) devuelve myString, a partir del índice que indique arg, hasta el final.
      • Ejemplo: myString.slice(4) devuelve ", ¿Qué tal?"
      • myString.slice(arg1, arg2) devuelve myString, a partir del índice que indique arg1, hasta el índice que indique arg2 (no incluído).
      • Ejemplo: myString.slice(0, 5) devuelve "Dijo"
    • Reemplazar partes de un String (replace(arg1, arg2))
      • myString.replace(arg1, arg2) devuelve myString, sustituyendo el string indicado en arg1, por el indicado en arg2.
      • Ejemplo: myString.replace("Hola", "Saludos") devuelve "Saludos, ¿Qué tal?"
